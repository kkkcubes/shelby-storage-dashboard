# Shelby Storage Dashboard

Decentralized file upload dashboard built with:

- Next.js
- Aptos Wallet Adapter
- Shelby Storage
- Socket.IO
- Express.js
- Tailwind CSS

## Features

- Wallet authentication
- Chunked uploads
- Upload progress tracking
- Video preview
- Real-time sockets
- Modern dashboard UI

## Frontend

```bash
cd frontend
npm install
npm run dev

## Backend

'''bash
cd backend
npm install
node server.js
